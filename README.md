comerciojusto_app
=================

App móvil del callejero de comercio justo en Vitoria/Gasteiz desarrolada para Medicus Mundi Alava.

La aplicación ha sido desaarolla usando los isguientes elementos:

 * Apache Cordova 4.0
 * Onsen
 * Jquery
 * Leaflet
